# 🎓 StudyFocusAI
StudyFocusAI is an AI-powered productivity tool designed to help students generate automated study plans, summarize complex notes, and create practice quizzes using NLP.

🚀 How to Run the Project
Follow these simple steps to launch the application:

1. Open the Project Folder
Locate the StudyFocusAI folder on your computer and open it so you can see all the files (like app.py).

2. Open the Command Prompt
Click on the address bar at the top of the file explorer window (where the folder path is shown), type cmd, and press Enter. This will open the terminal directly in that location.

3. Launch the Application
In the black terminal window that appears, type python app.py and press Enter.

4. Access in Your Browser
Once the terminal indicates the server is running, open your web browser and go to:

http://127.0.0.1:5000

💡 Key Features
Study Planner: Enter your subjects to receive a tailored timeline.

AI Summarizer: Instantly turn long notes into key bullet points.

Quiz Generator: Test your knowledge with AI-generated questions.
